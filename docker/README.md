## English

### Project Overview

A simple Docker implementation that runs an Nginx web server serving a custom HTML page with port mapping.

### What I Did

- Created a **Dockerfile** based on the official Nginx image
- Copied a simple **HTML file** into the container
- **Mapped port 8080** of the host to port 80 of the container
- Used **Docker Compose** to define and run the Nginx service
- Verified the setup by accessing `http://localhost:8080`
